# README



一种有趣的自我介绍或展示内容的方式, 适合不太严肃的场合.

没有 PPT 模板精美, 只是一种创意的形式.

可以运行 "个人介绍.ahk" 后, 如下操作查看效果.

---

输入 "个人介绍" 后自动补全.

按 Alt + 2 补全第二点. (Alt + 3 / 4 同样)

依次输入 "科普", "数学", "开心", "可爱", "好看" 均可自动补全.

输入 Alt + 5 补全结束语 1, 并显示第一张图片.

输入 Alt + 6 补全结束语 2, 并显示第二张图片.

---

可以根据需求修改内容, 展示其它内容.
